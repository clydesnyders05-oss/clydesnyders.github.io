# Clyde Snyders - Personal Portfolio Website

Welcome to the personal portfolio of Clyde Snyders, a Grade 9 student at Paterson High School (PHS) in the Eastern Cape, South Africa. This portfolio showcases Clyde's academic achievements, STEM projects, and personal milestones.

## Features

- **Responsive Design**: Fully mobile-friendly with modern UI components.
- **Hero Section**: Highlights Clyde's profile with dynamic visuals.
- **About Section**: An overview of Clyde's journey and core skills.
- **Achievements**: Includes key accomplishments in robotics and science fairs.
- **Gallery**: Interactive showcase of event and award photos.
- **Blog**: Insights into Clyde's experiences and perspectives.
- **Timeline**: Visual representation of academic and extracurricular milestones.
- **Contact Form**: Simple setup for users to reach out.

## Technology Stack

- **HTML**: Semantic structure for content.
- **CSS**: Styling with custom variables for themes.
- **JavaScript**: Functionality including animations, navigation, and interactivity.
- **Libraries**:
  - [Tailwind CSS](https://tailwindcss.com): Utility-first CSS framework.
  - [Animate on Scroll (AOS)](https://michalsnik.github.io/aos): For animations on scrolling.
  - [Three.js](https://threejs.org): Enhancing visual elements on the hero section.

## Setup and Deployment

To set up the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/clydesnyders05-oss/clydesnyders.github.io.git
   ```
2. Open `index.html` in a browser to view the website.

Alternatively, view the live website on [GitHub Pages](https://clydesnyders05-oss.github.io/).
